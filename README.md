# BoneVPython

Create a bone with vPython 

# How to use

**Create a instance**

    osso = Bone()
    
**Define your size**

    osso.setSize(size)
    
**Define your width and height**

    osso.setPos(width, height)

**Define your vertical and horizontal position**

    osso.rotate(vertical, horizontal)
    
**And generate**

    osso.generate()
    
*See an [example](sample.py)*
